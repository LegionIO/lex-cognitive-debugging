# lex-cognitive-debugging

Self-debugging system for cognitive processes in LegionIO.

Detects reasoning errors (inconsistencies, circular logic, overconfidence, etc.), traces the causal chain of how an error propagated through cognitive phases, and applies corrective strategies. Like a debugger for thought processes.

## Error Types

`:inconsistency`, `:circular_logic`, `:ungrounded_claim`, `:overconfidence`, `:logical_fallacy`, `:missing_evidence`, `:false_analogy`, `:confirmation_bias`

## Correction Strategies

`:retrace`, `:reframe`, `:weaken_confidence`, `:seek_evidence`, `:decompose`, `:analogize`, `:devil_advocate`
